# 如何同步fork #
## 流程 ##

## 1.进入你自己fork的项目 ##

![图片](pictures/如何同步fork/20160204111528.png)

## 2.点击 New pull request 按钮 ##

![图片](pictures/如何同步fork/20160204112637.png)

## 3.点击下方 switching the base ##

![图片](pictures/如何同步fork/20160204115847.png)

## 4.如果主项目有其他人提交改动,会出现红框内的内容,点击Create new request ##

![图片](pictures/如何同步fork/20160204120153.png)

## 5.填写内容,点击 Create new request 按钮 ##

![图片](pictures/如何同步fork/20160204125721.png)

## 6.出现如下图界面,点击 Merge pull request 按钮 ##

![图片](pictures/如何同步fork/20160204130534.png)

## 6.如果成功,出现如下图红框内容 ##

![图片](pictures/如何同步fork/20160204131115.png)
